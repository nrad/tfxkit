Usage Guide
===========

This guide explains how to use `tfxkit` for training and tuning.

Prequisites
------------
- Python 3.9+

Installation
------------

Install from source:

.. code-block:: bash

   git clone git@github.com:nrad/tfxkit.git
   cd tfxkit
   pip install -e .

Basic Workflow
--------------

.. code-block:: bash

   tfxkit 


1. Define your config file.
2. Select a model builder.
3. Run training or tuning.
